## 📂 Projetos Desenvolvidos

### 📌 Projeto — Sistema de Monitoramento de Colheitas

**Descrição:**
Este projeto tem como objetivo monitorar perdas na colheita de cana-de-açúcar, permitindo o cadastro de dados como produção esperada e real, cálculo automático de perdas percentuais e armazenamento das informações em arquivo JSON.

O sistema auxilia na identificação de prejuízos, contribuindo para a tomada de decisão no agronegócio.

---

**Tecnologias utilizadas:**

* Python
* JSON

---

**Principais aprendizados:**

* Criação de funções com parâmetros
* Uso de listas e dicionários
* Manipulação de arquivos JSON
* Validação de entrada de dados
* Estruturação de sistemas com menu interativo
